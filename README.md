# MAL No Total Episodes Extension
> [!WARNING]
> Better name needed :(

A really, really simple web extension that hides total episode count on the following pages:
* Anime info page (myanimelist.net/anime/*)
* User profile page
* Friend status (main page and myanimelist.net/history/*)
* User's anime list page
* Top anime page
* Anime search page
* Seasonal anime page
